# 3dsfont

## Usage

- ffnt2cfnt
~~~
ffnt2cfnt font.bffnt font.bcfnt
~~~

- font2charset
~~~
font2charset font.bcfnt charset.txt
~~~

- charset2xlor
~~~
charset2xlor charset.txt font.xlor
~~~

- charset2xllt
~~~
charset2xllt charset.txt font.xllt
~~~

- charset2xggp
~~~
charset2xggp charset.txt font.xggp
~~~
