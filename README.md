# 3dsfont

## Usage

- bcfnt2charset
~~~
bcfnt2charset font.bcfnt charset.txt
~~~

- bffnt2charset
~~~
bffnt2charset font.bffnt charset.txt
~~~

- charset2xlor
~~~
charset2xlor charset.txt font.xlor
~~~

- charset2xllt
~~~
charset2xllt charset.txt font.xllt
~~~

- charset2xggp
~~~
charset2xggp charset.txt font.xggp
~~~
